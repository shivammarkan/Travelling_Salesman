# Travelling Salesman Problem: An Interactive Educational Tool

An elegant, interactive web application designed to teach the **Travelling Salesman Problem (TSP)**, a classic challenge in computer science and combinatorial optimisation.

## Features

-   **Interactive Canvas**: Click to place cities and create your own TSP instances.
-   **Algorithm Comparisons**:
    -   **Brute Force**: Exhaustive search for the absolute shortest path (best for small sets).
    -   **Nearest Neighbour**: A fast, greedy heuristic that mimics human intuition.
    -   **2-Opt (Optimisation)**: An iterative improvement algorithm that "uncrosses" the path.
-   **Educational Mode**: Learn about NP-hard problems, factorial complexity ($O(n!)$), and real-world applications in logistics and manufacturing.
-   **Dual Theme Support**: Dedicated toggle for **Light** and **Dark** modes.
-   **UK English**: Built with British English standards for educational contexts.

## How to Run

Since this is a vanilla HTML/CSS/JS application, you can simply open `index.html` in any modern web browser or host it via **GitHub Pages**.

1.  Clone the repository.
2.  Open `index.html`.
3.  Add cities by clicking on the canvas.
4.  Select an algorithm to see it in action!

## Technology Stack

-   **HTML5 & CSS3**: Responsive layout and glassmorphism UI.
-   **Vanilla JavaScript**: Core logic and canvas rendering.
-   **Local Storage**: Remembers your preferred theme.
